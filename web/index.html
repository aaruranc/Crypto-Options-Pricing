Options-Pricer

Options-Pricer is an automated backtesting software. The premise is simple: you pick an asset you want option pricing analysis on and O-P computes the results and provides you a high-level statistical overview of historical option behavior. Upload a .csv file with 2 columns labeled 'Date' and 'Price' with time series data in the rows below. Please make sure that initial parameters are correct.


Asset Name Date Format Start Date End Date Trading Days
