Option-Backtester


Option-Backtester is an an exploratory data visualization tool. The premise is simple. You choose an asset that you want option pricing analysis on and O-B computes the results and renders them graphically. Upload a .csv file with 2 columns labeled 'Date' and 'Price' with time series data in the rows below. Please make sure that the start and end dates match the 'mm-dd-yyyy' format.

O-P only supports option pricing from 1/2/01 to present.
